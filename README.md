# WaterMark with TKinter

An App that allows user to watermark your uploaded images. 

### Activate venv

~~~    
    pip install virtualenv
    cd PATH/
    virtualenv venv
    source venv/bin/activate
~~~

### Run App

~~~
    python -m main.py
~~~
